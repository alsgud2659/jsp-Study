형상 관리
	-SVN
		서버에 있는 데이터를 모든 유저가 직접 공유한다.
		공유하는 절차가 쉽기 때문에 초보자도 쉽게 공유할 수 있다.
		하지만 서버에 문제가 발생하면 복구하기 쉽지 않다(백업파일이 없다면 복구 불가)

	-GIT
		서버에 있는 데이터를 유저별 각 서버에서 공유를 한다.
		공유하는 절차가 까다롭고 복잡하기 때문에 사전지식이 필요하다.
		하지만 서버 문제 발생 시 정상작동되는 한 명의 유저의 서버를 커밋하게 되면
		즉각 복구가 가능해진다.

1. 팀장이 svnTest 프로젝트에 jsp파일 한 개 생성 후 커밋
2. 팀원들은 svnTest 프로젝트 체크아웃
3. 순서를 정해서 팀원 한 명씩 본인 이름을 jsp파일에 작성 후 커밋
4. 모두 커밋했다면 팀장이 체크아웃 후 각 조원 이름 확인

1번 홍길동 -> 체크아웃 -> 본인이름 작성 -> 커밋
2번 이순신 -> 체크아웃 -> 본인이름 작성 -> 커밋
3번 둘리 -> 체크아웃 -> 본인이름 작성 -> 커밋
4번 짱구 -> 체크아웃 -> 본인이름 작성 -> 커밋
팀장 : 또치 -> 체크아웃
